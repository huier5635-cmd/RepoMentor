# RepoMentor 提交地址清单

提交前把占位符替换成真实地址。不要把 `127.0.0.1` 或 `localhost` 当作在线 Demo 地址。

| 项目 | 地址 / 命令 |
| --- | --- |
| Online Demo Frontend URL | `https://your-frontend-domain.vercel.app` |
| Backend Health URL | `https://your-backend-domain.onrender.com/api/health` |
| GitHub Repository URL | `https://github.com/your-name/repomentor` |
| Demo Video URL | `https://your-video-link` |
| PPT URL | `https://your-ppt-link` |
| Docker Run Command | `docker compose up --build` |
